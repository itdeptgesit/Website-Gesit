# Directive: Frontend Design Overhaul

## Objective
Implement a distinctive, production-grade visual overhaul for the Gesit Companies website, specifically for all "Our Business" pages (Property, Manufacturing, Trading & Services, Natural Resources), adhering to the "Refined Editorial" aesthetic.

## Inputs
- `src/app/(public)/our-business/property/page.jsx`
- `src/app/(public)/our-business/manufacturing/page.jsx`
- `src/app/(public)/our-business/trading-services/page.jsx`
- `src/app/(public)/our-business/natural-resources/page.jsx`
- `skill.md` (Design guidelines)
- Corporate brand colors: Blue (#103065), Gold (#BC9C33)

## Workflow
1. **Analyze Current Layout**: Evaluate the Property Page for generic patterns and grid-based rigidity.
2. **Apply Editorial Patterns**:
    - Introduce asymetrical layouts and overlapping elements.
    - Implement staggered motion reveals using `framer-motion`.
    - Enhance typography contrast and spacing.
3. **Refine Details**: Add subtle textures and layered transparencies.
4. **Validate**: Ensure mobile responsiveness remains high-quality after the overhaul.

## Authorized Tools
- `execution/style_optimizer.py` (if created) or direct file editing tools.

## Expected Output
- A visually striking Property Page that feels "designed" and "unforgettable".
- Consistency with the corporate identity while breaking the "standard corporate" mold.
